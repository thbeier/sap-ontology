SAP Ontology — sap: namespace

This IRI (https://thbeier.github.io/sap-ontology/vocab#) is the namespace for the SAP Ontology upper model, v0.1.1.

Class and property IRIs have the form sap:ClassName or sap:propertyName. For example, sap:ApplicationComponent resolves to https://thbeier.github.io/sap-ontology/vocab#ApplicationComponent.

Definitions

Class and property definitions live in JSON-LD schema files:

  • Architecture domain — ApplicationComponent, Integration, TechnologyBuildingBlock, DataObject
  • Process domain — Process, Activity, Event, Decision, BusinessDocument
  • Implementation/Change domain — Configuration, Change, Transport, TestCase, Incident, Requirement
  • Organization cross-cutter — OrgUnit, Role, Capability, User
  • Scenario cross-cutter — Scenario, ScenarioDiff
  • Provenance
  • Inter-domain relations
